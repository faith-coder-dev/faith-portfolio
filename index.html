SendIT Parcel Delivery App

Full-stack courier platform for parcel orders, tracking, status updates, and admin management.

SendIT Parcel Delivery App

Description: SENDIT is a full-stack courier service web app where users create parcel delivery orders, track parcel movement, and receive updates, while admins manage parcel statuses and locations.

Frontend: Next.js, React, TypeScript, Tailwind CSS, Shadcn/ui, Radix UI (deployed on Vercel)

Backend: Flask, Flask-RESTful, SQLAlchemy, Alembic, JWT, bcrypt (deployment to Koyeb in progress)

Database: PostgreSQL on Supabase

Features:

  • User authentication and parcel order creation
  • Order details view, destination update, and cancellation (before delivery)
  • Admin control for parcel status and present location updates
  • Map-based tracking with pickup and destination locations
  • Real-time email notifications for status and location changes

Deployment Status: Frontend is live on Vercel; backend deployment to Koyeb is still being finalized.
